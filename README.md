# Test Samples

Коллекция статических HTML-страниц с различными тестовыми примерами для веба. Каждый пример демонстрирует определённое поведение или ситуацию, которая может возникнуть на реальных сайтах. Примеры полностью независимы и не требуют установки зависимостей или настройки окружения. Проект размещается на GitHub Pages и доступен для использования в тестировании и разработке.

## Примеры

### [Задержка загрузки элементов](delayed-elements/)

7 элементов загружаются последовательно с задержками от 5 до 80 секунд. Каждый элемент показывает обратный отсчёт до момента загрузки.

### [Множество одинаковых элементов](duplicate-elements/)

Страница с множеством кнопок "Удалить" разных размеров и форм, но с одинаковым текстом. Включает таблицу с кнопками удаления в столбце и контролы управления над таблицей с иконкой корзины.

### [Вариации полей ввода](input-variations/)

30 различных дизайнов полей для ввода: от стандартных с рамками до минималистичных без границ, с лейблами в разных позициях, иконками, подписями и необычными стилями.

### [Поле ввода в стиле Draft.js](draft-editor-input/)

Пример поля ввода, реализованного через contenteditable div, как в редакторе Draft.js. Поле автоматически расширяется по высоте при вводе текста и поддерживает перенос слов.

### [Примеры дат и времени](datetime-shifts/)

Страница с различными примерами дат и времени с разными сдвигами: текущая дата и время, сдвиги на дни, часы, месяцы и годы вперёд и назад. Каждый пример подписан с указанием направления и величины сдвига.

### [Вариации шрифтов](font-variations/)

Страница с фразой "съешь еще съешь же еще этих мягких французских булок да выпить чаю", написанной 20 раз разными шрифтами. Под блоком находится большое текстовое поле для вывода всех вариаций.

### [Фиксированные и динамические переменные времени](time-variables/)

Страница с 4 парами полей для тестирования фиксированных и динамических переменных времени. Фиксированная переменная вычисляется один раз в начале таскейса и остается неизменной, динамическая обновляется в начале каждого шага.

### [Последовательности и сравнения](sequence-comparison/)

Страница для проверки понимания последовательностей и сравнений "больше-меньше". Включает кнопки разного размера по порядку, одинаковые кнопки в разных позициях, объект с кнопками слева и справа, и последовательность карточек с интерактивным выбором.

### [Позиции в углах](corner-positions/)

Страница с кнопками, расположенными в углах контейнера. Все кнопки имеют одинаковый текст, различаются только по позиции: левый верхний угол, правый верхний угол, левый нижний угол, правый нижний угол.

### [Стороны света](compass-directions/)

Страница с центральным объектом и кнопками, расположенными по сторонам света относительно центра. Все кнопки одинаковые, различаются только по направлению: север, юг, восток, запад, северо-восток, юго-запад и т.д.

### [Относительные позиции](relative-positions/)

Страница с примерами относительного позиционирования элементов. Кнопки расположены относительно центрального объекта: слева, справа, выше, ниже, слева выше, справа ниже. Все кнопки имеют одинаковый текст.

### [Позиции в сетке](grid-positions/)

Страница с сеткой 4x4 элементов. Все элементы имеют одинаковый текст, различаются только по позиции в сетке: первая строка, второй столбец, третья строка и т.д. При клике элемент меняет цвет.
